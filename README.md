# Order Pizza online

#### A program that allows users create their own pizza, see the price, pay and see confirmation of the purchase. 4/12/19

#### By **Maryana Antonyuk**

## Description

A website created with JavaScript and HTML, styled with CSS allows user to create their own pizza online and order it.


### Specs
| Spec | Input | Output |
| :-------------     | :------------- | :------------- |
| **Homepage** |  | Homepage with user input form |
| **Program Gathers User Selection** | User Select: sauce, crust etc  | Output: selected choice od sauce, crust |
| **Program Gathers User Selection and Calculate its price**| User Select: "thin crust" | Output: "thin crust, $5" |
| **Program offers a summary of selected choices and price and offers payment option**| Input: "size, sauce etc" | Output: summary of selected items, price and button "pay now" |
| **Program takes to 'Thank You' page after payment is successful** | Input: press button 'Pay Now'| Output: Page takes to 'Thank you' message and new page|

2. Clone this repository.
3. Open the command line--I use Terminal--type 'git clone' and repository link. 
4. You will have access to all files. Open index.html in your Internet browser. Enjoy!
